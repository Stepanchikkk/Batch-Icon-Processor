# Batch Icon Processor

Массовая обработка иконок с удалением подложки по эталону.

## Возможности

- Пакетная обработка тысяч файлов
- Удаление подложки по эталонному шаблону
- Превью оригинала и результата
- Модальный просмотр для сравнения
- Настраиваемые параметры чувствительности
- Экспорт результатов в ZIP
- Drag & Drop файлов
- Работает полностью в браузере

## Установка

```bash
npm install
npm run dev
```

## Деплой

Проект автоматически деплоится на GitHub Pages через GitHub Actions.

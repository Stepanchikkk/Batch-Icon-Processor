# Batch Icon Processor

Инструмент для пакетной обработки иконок: удаление фона по эталону и наложение нового.

## Возможности

**Шаг 1: Удаление фона**
- Загрузите эталонную подложку (чистый фон без иконки)
- Добавьте иконки для обработки
- Всё что совпадает с эталоном — удаляется

**Шаг 2: Наложение нового фона**
- Загрузите новый фон
- Добавьте иконки без фона (из шага 1 или свои)
- Настройте размер и масштаб
- Получите готовые иконки

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Готовый файл: `dist/index.html`

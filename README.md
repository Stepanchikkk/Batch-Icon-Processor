# Batch Icon Processor

Массовое удаление подложки с иконок по эталону.

## Демо

**[Открыть приложение](https://YOUR_USERNAME.github.io/batch-icon-processor/)**

## Как использовать

1. Загрузите **эталон** — чистую подложку БЕЗ иконки (обводка будет удалена автоматически)
2. Загрузите иконки (drag & drop или выбор файлов)
3. Настройте порог (20-40 для начала)
4. Нажмите "Обработать всё"
5. Скачайте ZIP

## Локальный запуск

```bash
npm install
npm run dev
```

## GitHub Pages

```bash
npm run build
# Загрузите папку dist/ на GitHub Pages
```

Или используйте GitHub Actions — создайте `.github/workflows/deploy.yml`
